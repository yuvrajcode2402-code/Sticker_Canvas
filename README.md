# Sticker Canvas

A mini React + Konva app where you can drop, drag, and download emoji stickers on a canvas.

## Features
- Click a sticker button to drop it on the canvas
- Drag stickers anywhere on the canvas
- Double-click a sticker to delete it
- Download the canvas as a PNG

## Getting Started

```bash
npm i
npm start
```

Opens at [http://localhost:3000](https://yuvrajcode2402-code.github.io/Sticker_Canvas)

## Tech Used
- React (functional components + hooks)
- react-konva for canvas rendering
